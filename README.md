# Bootcampspot CLI

## Installation

```
go get github.com/imander/bootcampspot/...
```

## Examples

### Configure Environment

```
$ bootcampspot configure
```

### Get Enrollments

```
$ bootcampspot get enrollments
```
 
### Get Weekly Feedback

```
$ bootcampspot get feedback
```

or specify a course ID

```
$ bootcampspot get feedback -c 1234
```
